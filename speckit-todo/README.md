# speckit-todo

Configured with a Spec-Kit-style workflow structure and implemented a skeuomorphic Todo app.

## Workflow artifacts
- `.specify/memory/constitution.md`
- `specs/001-skeuomorphic-todolist/spec.md`
- `specs/001-skeuomorphic-todolist/plan.md`
- `specs/001-skeuomorphic-todolist/tasks.md`

## App
- `app/index.html`
- `app/styles.css`
- `app/script.js`

## Run
```bash
cd speckit-todo/app
npx --yes http-server . -p 8091
```
Then open: `http://127.0.0.1:8091`
